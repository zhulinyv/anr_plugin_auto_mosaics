# 适用于 ANR 的自动打码插件

支持三种检测方法, 五种打码方法, 四个隐私部位检测. 简单易上手.

## 部分说明

1. 首次启动默认为 YOLO 检测.

2. 克隆本仓库后仅包含 YOLO 模型, 不包含 SAM 模型, 选用 YOLO+SAM 检测方法后, SAM 模型会自动下载.

3. sam_vit_h_xxxxxx 模型最大, 精度最高, 对 GPU/CPU 性能要求较高. sam_vir_b_xxxxxx 模型最小, 占用较小.

4. 若使用 YOLO+SAM 检测方法, 可以手动安装 CUDA 版本的 PyTorch 使用 GPU 对图片进行检测, 速度更快.

5. 关于线条马赛克, 会自动适应图片亮度选择黑色或白色线条.

## 未来计划

1. 支持自定义 YOLO 模型.

2. 支持视频打码.
